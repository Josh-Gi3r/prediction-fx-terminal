# FX Terminal desktop QA pass — 2026-06-10 (self-detected + fixed)

Reviewed every desktop page/section by screenshot. Findings + fixes:

## Fixed
1. **[SYSTEMIC] Headlines rendered white-on-white** — every `.ds4` page hero
   (home/earn/portfolio/trade + WC mini-heroes) showed faint/invisible
   headlines. Root cause: the design's `body{color:--ink}` reset became
   `.ds4 body` (a dead selector) when Phase-0 wrapped the CSS, so `.ds4`
   descendants inherited the legacy global `body{color:white}`. Fix: set
   `color/background/font` on the `.ds4` root directly (app/design.css). One
   line fixed the contrast across all pages — biggest win.
2. **Sawtooth price charts** (market detail + specials marquee/cards) — two bugs:
   (a) plotted raw Polymarket trades, which mix YES (~0.94) and NO (~0.06)
   prints → normalize No → 1−price for one YES series; (b) the SVG auto-scaled
   Y to the data min/max, so a 1¢ oscillation (93↔94) stretched to full height
   → floor the visual range (0.15 / 14pts) and centre. Charts now read as real
   near-flat price lines.
